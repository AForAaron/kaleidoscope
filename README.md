# 万花筒 · Kaleidoscope

一款运行在浏览器中的交互式万花筒，支持**现代数字**与**传统玻璃**两种风格，为手机和桌面双端设计。

---

## 项目概览

| 属性 | 详情 |
|------|------|
| 技术栈 | 纯 HTML + Canvas 2D，零依赖 |
| 文件结构 | 单文件（`index.html`），开箱即用 |
| 适配平台 | iOS Safari / Android Chrome / 桌面浏览器 |
| 对称扇区 | 12 折对称（12 个扇区） |

---

## 功能特性

### ✦ 数字模式（现代风格）
- **自由涂鸦**：用手指或鼠标在画面上拖动，笔触自动被映射到对称扇区，实时生成万花筒花纹
- **漂浮粒子**：圆点、星形、混搭三种粒子在图案中自动飘动
- **点击换色**：轻触画布（不拖动）快速切换配色方案
- **自动旋转**：图案持续缓慢旋转
- **陀螺仪倾斜**：倾斜手机可以旋转图案（需授权）

### ◆ 传统模式（玻璃片风格）
- **彩色玻璃片**：三角形、菱形、五边形等不规则多边形，带有半透明渐变和玻璃高光效果
- **完全静止**：玻璃片默认纹丝不动，真实还原传统万花筒放在桌上时的外观
- **拖动转筒**：在画面上拖动手指 / 鼠标，模拟转动万花筒筒身，图案跟随旋转并带有惯性衰减
- **晃动变换**：摇晃手机（DeviceMotion 超过阈值）或在桌面端点击画布，触发碎片翻滚重排（约 1.4 秒动画后静止）
- **8 种图案预设**：可在面板中选择不同形状的玻璃片组合
- **望远镜暗角**：边缘渐变遮罩模拟从圆形筒内观看的效果
- **陀螺仪旋转**：倾斜手机同样可以旋转图案（同数字模式）

## 技术实现

### 核心渲染逻辑

```
每一帧：
1. 清空画布，填充背景色
2. 将坐标系原点移至画布中心
3. 循环 12 个扇区：
   a. 旋转到当前扇区角度
   b. 奇数扇区沿 Y 轴镜像（形成对称）
   c. 裁剪到楔形区域
   d. 绘制内容（粒子 / 玻璃片）
4. [传统模式] 叠加暗角渐变遮罩
```

### 笔触映射（数字模式）

用户触摸坐标 → 转换为极坐标 → 对角度取模（折叠到第一个半扇区）→ 绘制在离屏画布 → 每帧复制到所有扇区

### 玻璃片渲染（传统模式）

每个玻璃片：多边形路径 + 线性渐变填充（亮角 → 基础色 → 暗边）+ 白色边缘高光 + 椭圆反光点

### 晃动检测（传统模式）

```javascript
DeviceMotionEvent.acceleration → 
  Math.sqrt(ax² + ay² + az²) > 阈值 →
    触发碎片重排动画
```

## 本地运行

直接用浏览器打开 `index.html` 即可（无需服务器、无需安装）：

```bash
# macOS
open index.html

# 或通过本地服务器（推荐，避免某些浏览器的文件协议限制）
python3 -m http.server 8080
# 然后访问 http://localhost:8080
```

**iOS Safari 陀螺仪**：首次点击"开始"按钮时会弹出权限请求，允许后即可使用倾斜和晃动功能。
